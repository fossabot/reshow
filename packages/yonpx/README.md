# `yonpx`

A shorter for 
> `npx -p yo -p generator-app -c 'yo app'` to 
> `npx yonpx app`

## Repository 
* `GIT`
   * https://github.com/react-atomic/reshow/tree/main/packages/yonpx 
* `NPM`
   * https://www.npmjs.com/package/yonpx

## Usage

```
npx yonpx [your-generator] your-app-name
```

ex:

```
npx yonpx reshow myapp
```
